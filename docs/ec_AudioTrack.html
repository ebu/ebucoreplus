dcterms:description
Une audioTrack est le conteneur de données audio de base d'un support. L'attribut est une référence non ambiguë à ce conteneur dans un support donné.
Ein audioTrack ist der grundlegende Audiodaten-Container eines Mediums. Ein Attribut ist ein eindeutiger Verweis auf diesen Container in einem bestimmten Medium.
An audioTrack is the basic audio data container of a medium. Attribute is an unambiguous reference to this container in a given medium. An audioTrack object defines a component of an audioStream. A single set of samples or data in the storage medium. Represents a physical container or carrier to hold an audio stream.

Class 'Date-time description'

dcterms:description
Description of date and time structured with separate values for the various elements of a calendar-clock system. The temporal reference system is fixed to Gregorian Calendar, and the range of year, month, day properties restricted to corresponding XML Schema types xsd:gYear, xsd:gMonth and xsd:gDay, respectively.

Beschreibung von Datum und Uhrzeit, strukturiert mit separaten Werten für die verschiedenen Elemente eines Kalender-Uhr-Systems. Das zeitliche Bezugssystem ist auf den Gregorianischen Kalender festgelegt und der Bereich der Eigenschaften Jahr, Monat und Tag ist auf die entsprechenden XML-Schema-Typen xsd:gYear, xsd:gMonth bzw. xsd:gDay beschränkt.
